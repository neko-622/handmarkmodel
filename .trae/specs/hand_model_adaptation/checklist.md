- [ ] 检查模型加载是否成功：程序启动时应无模型加载错误
- [ ] 检查手部检测是否正常：输入包含手部的图像时应能检测到手部
- [ ] 检查手部关键点标注是否正确：检测到手部后应能正确标注关键点
- [ ] 检查可视化是否正常：OSD上应能显示手部检测框和关键点标注
- [ ] 检查性能是否满足要求：程序运行时应无明显卡顿
- [ ] 检查代码编译是否成功：修改后的代码应能成功编译
- [ ] 检查代码结构是否清晰：代码应保持清晰、注释充分
- [ ] 检查模型路径是否正确：确保程序使用的是cut_model.onnx模型
- [ ] 检查输入输出格式是否正确：确保模型的输入输出格式处理正确
- [ ] 检查后处理逻辑是否正确：确保手部检测框和关键点的提取逻辑正确
